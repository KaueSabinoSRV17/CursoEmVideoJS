Como podemos usar o Javascpript

Podemos iniciar o uso do Javascpript de duas maneiras principais: podemos usa-lo dentro do próprio HTML, através da tag script (nao é recomendado) ou podemos fazer da forma mais apropriada, que é inserindo uma tag script:src, onde pegamos um script vindo de um arquivo externo

Comandos Básicos

Para começar no Javascpript, podemos usar os seguintes códigos:

  • Alert

    window.alert('frase que vai aparecer na caixa de alerta') - Este código vai invocar uma caixa de alerta assim que a página for carregada, com uma mensagem que pode ser escolhida pelo programador.

  • Confirm

    window.confirm('frase que vai aparecer na caixa de alerta') - Este código vai invocar uma caixa muito parecida, mas com um botao de "ok" e de "cancelar"

  • Prompt

    window.prompt('frase que vai aparecer na caixa de alerta') - Este código tem a funçao inédita de deixar o usuário escrever em um campo de texto, que pode ser armazenado pelo próprio programa. veremos mais como usar esta possibilidade no próximo Código

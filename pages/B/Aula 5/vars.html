Variáveis

As Variáveis são espaços para guardar dados e usa-los em um outro momento do código. para serem declaradas no JS, devemos usar a palavra var e atribuir qualquer valor com um "=" (apenas um igual, sozinho se lê SEMPRE como "recebe" no Javascpript). Por exemplo, podemos usar os comandos DOM já ensinados que manipulam específicamente a tabela, sem entrar na árvore (comandos window.")

Regras

Devemos usar as seguintes regras para declarar variáveis:

  • - Podem começar com letras, $ ou _
  • - NÃO podem comçar com números
  • - É possível usar letras ou números
  • - É permitido o uso de acentos e símbolos
  • - Nao podem conter espaços
  • - Não podem ser palavras reservadas a comandos da linguagem, como var ou function

Dicas

  • - Maúsculas e Minúsculas FAZEM diferença!
  • - Use nomes coerentes para as suas variáveis

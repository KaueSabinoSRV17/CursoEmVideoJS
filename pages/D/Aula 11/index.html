Condições em Javascript

Condições, em qualquer Linguagem Imperativa, é uma forma de declarar código a ser executado apenas após uma condição for atendida. Podemos ter condições:

    Simples

  • Acontecem quando apenas precisamos ter um código para o caso da condição ser verdadeira

    • Composta

  • Acontece quando precisamos de código para caso a condição também não seja atendida, sendo apenas um else ou um ninho de if e else

Variáveis Compostas

Uma variáveis composta é uma variável que armazena mais de um valor

Vetores

Um vetor é uma variável composta que armazena mais de um valor, onde estes valores ficam dentro dos elementos, que são acessados através de índices, onde o primeiro é 0

Comandos de vetores

Todo vetor é um objeto que tem métodos básicos. são alguns deles:

  • .push(valor a ser adicionado) - Método que adiciona o valor
  • .length - Atributo que armazena a quantidade de elementos do vetor
  • .sort()- Método que vai ordenar os elementos
  • .indexOf(valorProcurado)- Método que vai procurar o valor recebido como parâmetro e retornar sua chave de acesso no Vetor. caso este valor não exista neste Vetor, ele vai retornar -1

Integração do For

Para fazer diversas operações com os elementos de um vetor, podemos usar um For para percorrer este vetor e executar comandos referentes a cada elemento. inclusive, há um for simplificado e otimizado especialmente (e únicamente) para percorrer Variáveis Compostas, ele é assim: for(let i in variavel_composta) {comandos}
